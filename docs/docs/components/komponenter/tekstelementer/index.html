Tekstelementer

Tekstelementer show tabular data in columns and rows.

Headings

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

  • Preview size: 375px (Mobile device, ie. iPhone6)
  • Preview size: 768px (Tablet, ie. iPad)
  • Preview size: 1024px (Desktop / Laptop)
  • Preview size: 100% (Full Screen)

<h4>h4. Heading</h4>

Body copy

The Frontend styleguide's global default font-size is 14px, with a line-height of 1.428. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).
